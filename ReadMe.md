# D&D Trader — Генератор торговцев с предметами

Веб-приложение для Мастеров D&D: автоматически генерирует торговцев (кузнецов, алхимиков, трактирщиков) с ассортиментом, ценами, скидками от репутации. Есть корзина, инвентарь, учёт золота, резервирование, заметки ГМ, личный кабинет с файлами и картой.

**Сайт (demo):** [https://dnd-trader.onrender.com](https://dnd-trader.onrender.com)

## Особенности

- 20+ уникальных торговцев с именами, типами, регионами, репутацией
- 2100+ предметов (оружие, броня, зелья, свитки, магические штуки) из базы dnd.su
- Цены зависят от репутации (скидка/наценка)
- Фильтры: по торговцу, предмету, цене, редкости, категории, магии
- Модалка торговца с вкладками: товары, продажа, статы, заметки ГМ
- Корзина и резервирование предметов
- Инвентарь игрока (продажа/удаление, добавление кастомных предметов)
- Звуки монетки и двери
- Аутентификация (регистрация/вход), сохранение персонажа на сервере
- Личный кабинет: история кампании, задания, карта, файлы, заметки ГМ
- Адаптивная вёрстка
- Лицензия CC BY-NC-SA 4.0 (некоммерческое использование)

## Технологии

- **Backend:** Python 3.10 + FastAPI
- **Database:** PostgreSQL 15
- **Frontend:** HTML, CSS, Vanilla JS
- **Containerization:** Docker, Docker Compose
- **Deploy:** Render.com (или любой VPS)

## Установка и запуск (локально)

### 1. Клонировать репозиторий

```bash
git clone https://github.com/AnderSunI/dnd-trader.git
cd dnd-trader
dnd-trader/
├── app/                     # бэкенд FastAPI
│   ├── main.py              # основной сервер, эндпоинты
│   ├── models.py            # SQLAlchemy модели (Trader, Item и др.)
│   ├── auth.py              # JWT аутентификация
│   └── seed_db.py           # список торговцев (20 штук)
├── frontend/                # статика
│   ├── index.html           # весь фронтенд (JS/CSS в одном файле)
│   ├── sounds/              # coin.mp3, door.mp3
│   └── images/              # background.jpg, портреты торговцев
├── cleaned_items.json       # нормализованные предметы (2178 записей)
├── requirements.txt         # зависимости Python
├── Dockerfile               # сборка образа
├── docker-compose.yml       # запуск app + postgres
├── .env.example             # пример переменных окружения
├── .gitignore
└── README.md
